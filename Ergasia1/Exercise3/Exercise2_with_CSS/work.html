Personal Projects

Some personal projects I worked on as preparation for certain courses

Javascript

In preparation of 2021 eastern semester, I knew Web Development would be one of courses I would be working and that JavaScript would be heavily involved, so while going through a couple of JS web courses I stumbled upon a video called "Developing a Discord Bot with JS".

Discord is a chat app, similar to programs such as Skype or TeamSpeak, or professional communications platforms like Slack.

It supports video calls, voice chat, and text, allowing users to get in touch however they please. It's a platform that I have been using daily since 2016 and has been a way to keep in touch with friends and colleagues during the pandemic.

I have been part of a Discord Server that frequently does "movie-nights" where members throughout the week post IMDB link's of a movie, moderators gather all the links and put up a voting system where users "react" using emojis to submit their vote. I thought a bot that would automate this procedure would be good introduction to JS and would be fairly quick to do.

I started by getting familiar with a few JS terms and then moved on setting up Visual Studio Code, NodeJS and the Discord API called DiscordJS, the whole project was finished in about 10 days and consisted of the following functions:

  • Invite links sent to the user through a private message
  • Handling movie submissions that would be put on vote at the end of week
  • When grades are published about a course the user is interested in, sent a private message to notify him

AutoHotKey

I need to state that I was never a fan of work and assignments that required the typing of thousands of words, and a big part of that dissatisfaction was the need of intonation which I never used in personal online conversations.

So I thought, why can't I make a simple script that will take a word like "καλημερα" and automatically turns it into "καλημέρα".

And so I did using a free open-source scripting language called AutoHotKey, I combined a simple script with a database I made of over 300.000 words.

The database contained the word with and without its intonation, and the script kept track of what typed until I pressed <space>, if the word existed in my database, it would replace it with the one with the right intonation.

I have been using it on my personal computer on daily basis and has proved really useful especially when writing text-based assignments or reports.
