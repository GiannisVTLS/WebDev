About Discord

An introduction to Discord and its API, Discord JS

Introduction

Discord is a popular group-chatting app that was released in 2015 to give gamers a place to build communities and talk.
Discord is free and more user friendly and has since branched out to include communities from all over the internet, ranging from writers to artists to programmers.
It saw its first rise to popularity in 2016 due to heavy advertisting in Twitch.tv and became widespread during the pandemic, as more people have worked, played games, and socialized online, the platform supports that it now has 100+ million monthly active users.

Discord Server Structure

Discord follows the model set by Slack that was released in 2013, but in a less formal way.
At a basic level, Discord is built to allow members to message each other by creating communities called servers, which can be either public or private(personal invite only).
Servers are created and maintained by individual users called moderators and are then filled with text channels, voice channels and since recently live streaming channels.
You can also share videos, images, internet links, music, and more.

Discord Bots

With the fast growth of Discord in popularity, it became apparent that moderators needed a way to automate certain actions in order to handle thousands of users since automated processes, such as banning inappropriate users and reacting to user requests are vital for a community to thrive and grow.
So Discord launched its API that supports multiple programming languages inluding C#, JavaScript, Lua, Python and more.
Now users can create automated programs, aka bots, that look and act like users and automatically respond to events and commands on Discord.

Discord JS

Discord.js defines itself like so:
"Discord.js is a powerful node.js module that allows you to interact with the Discord API very easily. It takes a much more object-oriented approach than most other JS Discord libraries, making your bot's code significantly tidier and easier to comprehend. Usability, consistency, and performance are key focuses of discord.js, and it also has nearly 100% coverage of the Discord API. It receives new Discord features shortly after they arrive in the API."
So the only prerequisites to creating a bot, is to have a fairly decent grasp of JavaScript and previous experience with an OOP programming language, so as to not get stuck on uncomplicated issues that will prove to be frustrating.
