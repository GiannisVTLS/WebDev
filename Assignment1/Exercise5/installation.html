Quick-Install Guide

Detailed guide on how to install and use DiscordJS

DiscordJS

Since DiscordJS is a node.js module, the first step is to install node.js. You can do so by going to the Node.js website.
Now that node.js is installed, getting DiscordJS is fairly trivial.
All you need to do is open your console and run the following command: npm install discord.js.
This might take a bit of time depending on your internet connection, but it should be fairly fast.

Creating a bot

There are a few more steps to complete before you can start coding though.
The next step you need to take is setting up an actual Discord bot application via Discord's website.
Thankfully Discord Devs have made it a fairly easy process. The steps you need to take are as follows:

  • Open the Discord developer portal and log into your account.
  • Click on the New Application button.
  • Enter a name and optionally select a team.
  • Then confirm the pop-up window by clicking the Create button.

Click the Add Bot button on the right and confirm the pop-up window by clicking Yes, do it!. The Discord Bot was now created and you now have to ability to customize some of its attributes before adding it to your server.

In the panel that was created, you can customize its profile picture, set its username, and set its visibility to either public or private.
From this window you can also access the token unique to your bot, either by revealing it or pressing the Copy button, this will be useful later on.
When we ask you to paste your token somewhere, this is the value that you need to put in.

Adding to Server

In order to add the bot on your server you will need to create a unique invite link and paste it into your browser.
One such link will look like this:
https://discord.com/oauth2/authorize?client_id=123456789012345678&scope=bot
While the process of creating one is fairly simple once you breakdown the sections of it:

  • The first part is just Discord's standard structure for authorizing an OAuth2 application (such as your bot application) for entry to a Discord server.
  • The second part that says client_id=...  is to specify which application you want to authorize. You'll need to replace this part with your client's ID to create a valid invite link.
  • Lastly, the third part, which says scope=bot , specifies that you want to add this application as a Discord bot.
